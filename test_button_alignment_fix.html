技术实现

🔧 关键技术点

  • HTML结构重构：替换整个按钮容器而不是追加
  • CSS优先级：使用!important确保样式生效
  • Flexbox布局：align-items: center确保垂直对齐
  • 统一类名：my-space-btn类统一所有按钮样式
CSS关键代码：
.my-space-btn {
  background: linear-gradient(...) !important;
  height: 32px;
  display: inline-flex;
  align-items: center;
}

.prompt-card-actions {
  display: flex;
  align-items: center;
  justify-content: flex-end;
}
